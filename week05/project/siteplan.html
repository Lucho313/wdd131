Website Planning Document

Site Name

Temple Explorer

This name was chosen because the website focuses on exploring temples around the world, providing key information and filtering options for users.

Optional Domain: templeexplorer.com

Site Purpose

The website provides users with information about different temples, including location, size, and dedication dates. It also allows users to filter temples based on different categories such as age and size.

Scenarios

  • Which temples are the oldest?
  • Which temples are the largest in size?

Color Schema

Primary Color (#002855): Used for headers and footer.

Accent Color (#f4a261): Used for buttons, highlights, and borders.

Background (#ffffff): Used for the main page background.

Typography

Playfair Display: Used for headings.

Roboto: Used for body text.
